# Ether Signal

A React implementation of https://github.com/vulcanize/ethersignal

## Setting up development server
From terminal:
```js
git clone git@github.com:langateam/ethersignal.git && cd ethersignal
npm install
npm start
```
Open localhost:8080 in MIST.
